# To-Do List Application

This is a simple To-Do List application built using TypeScript and DOM manipulation.

## Table of Contents

- [Features](#features)

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All, Active, Completed)
- Persist tasks in local storage
